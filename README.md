# Kora Status

📈 Uptime monitor and status page for [Kora AI Platform](https://kora.maxwi.com), powered by [Upptime](https://github.com/upptime/upptime).

**Live Status:** [status.kora.maxwi.com](https://status.kora.maxwi.com)

## Monitored Services

| Service | URL | Description |
|---------|-----|-------------|
| Kora Web | `kora.maxwi.com` | Main web application |
| Kora API | `api.kora.maxwi.com/health` | API Gateway health endpoint |
| Kora Hub | `kora.maxwi.com/hub` | Agent marketplace |

## How It Works

- GitHub Actions checks service availability **every 5 minutes**
- Response time graphs are generated **every 6 hours**
- Incidents are automatically created as **GitHub Issues**
- Status page is deployed to **GitHub Pages**
